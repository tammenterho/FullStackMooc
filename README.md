# Full Stack Mooc

The course covers modern web application development with JavaScript. The main focus is on single-page applications implemented using the React library, as well as REST APIs implemented with Node.js to support them. The course also introduces GraphQL as an alternative approach to API implementation compared to REST.

The course also addresses testing, configuration, and managing execution environments for applications, along with the use of databases for storing application data.
